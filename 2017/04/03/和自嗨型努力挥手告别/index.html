和自嗨型努力挥手告别

发表于 2017-04-03 | | 阅读次数

有些事,比努力更重要.

我认识的世界错了吗

三月,迎来了又一次职级晋升答辩,自我感觉信心满满,面对晋升标准,无论是业绩,还是能力模型,感觉匹配程度都比较高,结果却啪啪打脸,进了待定区.自我感觉还挺努力的,努力写代码,有任务就快速搞定.用什么学什么,一心把技术学起来,当一个手艺人.可结果并不是太好.似乎身边也有不少这样的人,明明很努力,结果却不尽人意,直观上这种处境特别像前两年那个梗,男生很努力的追求妹子,妹子表示很感动,但是拒绝了.感觉以前在学校的时候稍微用功就能有不错的效果,这到了工作怎么就不行了呢?是社会和学校的生存规则不一样吗？

成本并不能决定价格

这个问题先留下,先讨论另一个问题,北京的房租(房价我都无力吐槽),就拿我公司附近的房子来说,差不多的一居室要5000元/月.以前我一直在想,这北京消费水平也太高了.想住的离公司近一点,为什么这么贵?想到的第一个答案就是,房子贵.可都是房子,为什么这里的这么贵,家里的却不贵,看来问题不在房子,于是我觉得应该是地贵,可家里也有地,为什么家里的地却卖不出这种价格,看来答案也不在地.而思考这些的时候我竟然还忽略了一个事实,房子还在涨价,并且远超每年的通胀率!我之前下意识的觉得,东西贵,是因为成本高.可是成本一定的情况下,价格却在升高.这显然不符合成本决定价格.最近我才想明白,其实上学的时候老师就讲过,价格是由供求关系决定的,并不是成本.(想起来的时候瞬间觉得这么多年活的不够清楚,最难受的是自己都不知道自己活的不明白),为什么房地产商人会在北京花那么多钱,买下地.因为他们知道,买下地,建房子,会有人来买单的,在北京这个地方,人们就是会花大笔的钱,来买住房,需求在这里,跑不了.那他为什么不买地做别的,比如开一个商场.当然他也会有其他的考量,只不过最终选择盖房子卖楼.是因为这样相对其他其他方案可能盈利更多一些.

和我有什么关系

现在知道了价格是需求决定的,上学的时候就学过,可这有什么用?重新回到我自己,实际上我和那些建起来的房子是一样的.我的价值并不是我们自己决定的,而是社会(公司,部门)对我的需求决定.那我应该如何能提升我自己的价值,答案也就是让更多的人需要我,增强自己和这个客观世界的联系了吧.那回到最开始的问题,为什么我感觉自己挺努力,也确实还可以,最后结果不太好呢?因为我的努力,只是在满足自己的需求,而不是公司的,部门的.哪怕满足,也只是满足自己的同时,顺便有那么一点点契合,而这一点点契合,也是一定程度的巧合.我的着落点始终在,如何写出更好的代码,如何去学更多的东西,让自己更强,我一直认为定价的主动权在我.可是很遗憾游戏规则不是这样.部门不在乎你如何写代码,如何的去学习,不在乎你是要当一个极客或者是科学家.而部门要的是业务的发展,以及能够帮助业务发展和能为业务发展做更多事情的人.而不是一个只会自嗨的去写代码的程序员,一开始的关注点就错了,自然中间遇到选择的时候会做出不一样的决定,结果其实在一开始就注定了.哪怕这次运气好,撞上了,那下次呢,下下次呢.这本身就不是一锤子买卖,没人会一直走运的.在学校的时候,外界对大家的需求就是好好学习,考个好成绩,那时候你在做的事情和外界对你的需求高度吻合,所以你努力,结果自然很大程度上就会很好.可这并不代表你努力就足够,另一个隐性条件,也同样重要,甚至比努力更重要,那就是要挣扎着认清楚,你现在需要满足的需求是什么,而不仅仅是,你想要做什么.方向错了,可能一切的努力都是白费.

可以不这样做吗

可能有人会说,我为什么一定要想着满足他人的需求呢,我自己追求的就是开心,自由.不行吗?我觉得这当然没问题.我喜欢去研究代码怎么写更好看,更易读,扩展性更好.能写出高质量的代码让我感觉开心,这当然没问题,但是当我将注意力都放在这件事情.甚至我觉得我作为程序员,就应该把全部的关注点放在这个地方,而没有去考虑满足部门对更高层级的员工的真实需求(虽然说对技术也有一定的要求,但这只是占了不大的比例)的时候,我也就主动放弃了成为更高层员工的机会.更可怕的是,我自己并不知道我自己做了这么一个选择.不知道自己在放弃这样的机会,我没有考虑到真实世界对我的要求是这个样子的.所以哪怕我十分努力,然并卵.

配的上,真的不容易

想要得到某样东西的最可靠办法就是让自己配得上他,这是查理芒格的一句著名鸡汤.我在听罗胖解读”配得上”的时候觉得特别精彩,他说,配得上就要求我们提升自己的内在价值系统,去匹配我们想要的东西,才能认清楚他的价值.还是举个我自己的例子吧,基本每个人都想赚大钱,以前我看别人身价XXX,都会觉得,给我那么多钱就好了.我想要那么多钱,谁不想,不是吗.可我有一天又想,如果有一天真的给了我那么多钱,花不完的钱,我会在做什么,我生活的意义又会是什么,我会怎么花这笔钱,挥霍都挥霍不完的钱.我发现我不知道,我甚至都不知道在满足了我的各种欲求之后,要做什么.这么多钱怎么办.而那些真正有这笔钱的人,似乎没有这些疑问,他们知道该怎么做,他们有处理这么多钱的能力与智慧.他们配的上,而我现在配不上.所以,我现在也就自然没有那么多钱.

不仅如此,很多时候需求是不明确的,甚至连需求方自己都不知道自己想要什么.想想招聘网站的jd你就知道了.大家写的都差不多,要求X年工作经验,巴拉巴拉一大堆要求.可你真的需要全都满足吗,或者说你全都满足就会要你吗?显然不是.但这类需求至少我们可以通过不断的去试错,因为我们还是有一个大方向去准备尝试.我从大三的时候就开始不断的参加面试,也鼓励身边的朋友去参加.哪怕懂的很少,虽然他们最终也没什么人听我的去面.因为我发现只有通过不断的和真实世界去接触,你才能知道他们真正的需求,你觉得自己学的还不够,可学多少才算够,学到什么样子才算准备充分,你学的是他们想要的吗,你不知道.不过遗憾的是当时我没能想到今天想到的这些.如果想到了,也许现在又会是另一翻场景.

还有一种情况,连需求方都不知道自己的需求是什么,你可能觉得这怎么可能?这当然可能,没有微信的时候,你觉得有事情打打电话,聊聊qq也挺好,然后微信来了.你发现,我靠,这东西真是太好了.这几年发生的这种事情还少吗？这是需要多么强大的共情能力,才能将一个个用户都不知道自己有的需求,帮用户解决掉.所以,配得上,真的很难,不过再难也要挣扎着去做,因为那才是我真正想去的地方,那儿才有我想看到的风景.

没有总结的总结

文章没有结尾总结,因为我还没写完.我还有很多话题想聊,有很多字想写,但是觉得还不够通透,所以就先写到这吧.希望我的一些小思考,也能让看到的人产生一些新的想法,一点点的感触.
