接纳自己

发表于 2017-02-19 | | 阅读次数

这么多年过去了,你是什么熊样,你自己还不知道吗.

之前一直想把放在看盘上的注意力抽离出来做点别的事情,然而一个多月下来,试了很多办法,都没什么卵用.明明能感受到自己受到的影响,却没什么太好的改变效果,也是一度让我感觉很烦.并且第一次感受到,想做,但是做不到是一种什么样的体验.其实生活中这种场景很多,一般都可以靠一些其他的方法避开,比如游戏,删了就没事了.比如分神,多专注几次,多练习几次,自然也能专注的多.倒是像这种,白天还可以靠自制力维持,可到了下班的地铁,到了睡前的床,累了一天之后,简直难以抗拒诱惑,一刻满足过后,袭来的就是无法掌控自己的无力感.

我给这段时期起了个名字叫”认知升级阵痛期”,因为每次的认知升级,都会有这么一个时期.原因的话也很简单,人每一次的创造行为,实际上都可以分解为两次,一次是大脑创造,一次是真实的创造。而阵痛期,就发生在大脑创造已经完成,而真实创造却没有完成的时间段.其实举例来说很简单,每个人都知道自己有拖延症,也知道拖延症影响了一些生活,但是没卵用啊,还是在忍不住的拖延.偶尔心里还会产生负罪感.最近看了一篇关于拖延症的文章,全文主旨就四个字,接纳自己.简单来说,我知道短期没有办法改变,我只能一点点的改善,一点点的和这种感觉和睦相处,这其实是一件很有意思的事情.当下次想拖延的感觉来了,可以在心里默默的对另一个想拖延的自我说,hello,你又来了,然后你可以评估一下,现在是否允许自己稍微拖延一小下.或者说现在不可以拖延,一会再拖一下.实际上当你可以直面自己的内心,和另一个”自己”商量的时候,负面情绪会慢慢消散.

一生不长,认清自己,接纳自己,也许是快乐生活的另一把钥匙吧.
