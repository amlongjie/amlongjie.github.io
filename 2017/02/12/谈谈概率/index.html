谈谈概率

发表于 2017-02-12 | | 阅读次数

把概率这个工具放进你的工具箱.

赌徒谬误

投硬币连续出现n次正面之后,下次出现正面的概率是多少?答案是1/2.

学过概率的人都知道,投硬币是独立事件,前面的多少次结果都不会影响下一次发生的概率.这多少有点违背人的直觉,正常情况下,我们潜意识会觉得已经出现这么多次正面了,下次出现还是正面的概率应该很小,所以出现反面的概率应该很大.我们不讨论如何运用概率公式进行独立事件计算,我们分析为什么会有这种直觉,这种直觉错了么?假定H代表投硬币投出正面,T代表投硬币投出背面.如果我们连续投掷5次,从第一次开始计算,出现HHHHH的概率是(1/2)^5 = 3.125%, 现在已经投掷了5次,要投掷第六次,那我们从第一次开始投掷的角度去看,出现HHHHHH的概率是(1/2)^6=1.56%,而出现HHHHHT的概率也是1.56%.我们的潜意识在判断这个问题的时候,每次都是从第一次投掷开始分析,那连续出现6次H的概率必然小于连续出现5次H的概率.但是我们面对的问题并不是投五次和投六次的概率比较问题,而是,已经出现5次H了之后,下次投出H和T的概率比较,结果很明显,都是1.56%,第六次投掷,遇到正面反面的概率相等.

虽然只是一个小小的问题,却能折射出我们在思考中的思维方式,在这个问题的判断上,潜意识在使用绝对的角度去看待这个问题,并且由于这个错误的角度,导致一个不正确的因果推理,而实际上,在这里我们需要使用相对的角度去思考.

概率的意义

这世界上很难的一件事,就是让一个人在事前明白,他本可以拥有这件事.为什么难?因为还没有拥有,甚至从来没有拥有过.为什么要事前,因为事后就已经晚了呀.举例来说,每个人都知道我如果当初好好学习,早就考上了清华北大,走向人生巅峰.对,这种本可以拥有,是你事后清楚的意识到,曾经存在这种可能.可问题是,这种已经事后的醒悟,对我们来说,没什么指导意义.

再举个例子,保险.很多时候身边的人都会接到保险推销员的电话,但是多数的人听到是卖保险的,就匆匆挂断电话.人们可能会说,我还年轻,我这么健康等等理由.并且在若干年,可能十年内,确实他们也没有生过什么大病,出过什么事故,可是,在这十年里,他们却是实实在在的可能会面对这些疾病,意外.只不过,没有遇到.所以他们从来不会明白,本可能拥有,却没能拥有这件事(虽然这不是一件好事情).这可能也是保险不好卖的原因吧.当然就算有人知道要买保险,但是面对琳琅满足的保险险种,可能会无从下手(这种情况很常见),于是也就要么放弃,要么听从指导.这里需要说明,听从指导不是不好,就是有时候可能被宰,仅此而已.我个人的话比较喜欢掌握更多的主动权.

说了这么多,当然我今天不是来卖保险的.谈谈如何用概率的角度去分析这些事情吧.还是用保险的例子吧,假定现在有一个重疾险,六种疾病在被保列表中,那么我们应该怎么做,很简单,去检索数据,获取最几年,各个疾病在各个年龄段的发病率,假定六种分别是1%,2%,3%,4%,5%,6%.那六种疾病未来一年你都不会患的概率大约是81%,那未来十年都不会患的概率呢,0.81^10 = 12.1%.那么我们就有结论,未来十年有接近88%的概率会患病.那么,你现在是否真的不需要一份保险来保障一下呢.

人和人的差距有时候真的要比看上去要大的多,有的人从来就没考虑过原来很多东西是可以计算的.

总结

这篇文章对我来说,还是挺难写的,概率实际上我学的也不是很好,也只能做一些简单的运算,不过已经对我们的生活产生了不少的帮助,希望可以给看到的人一些启发吧.最近看到有意思的一句话,知识才是金钱,时间才是力量.2017年又过了一周,你的年度计划,是不是已经都忘到脑后了呢?
