个人奋斗只是一种自欺欺人吗

发表于 2017-02-05 | | 阅读次数

最近在和朋友吃饭的时候聊到一个有意思的话题.不断努力,走向成功不过是人的一种自我安慰.难道你努力的更多.你就比一个努力的更少的人更可能成功吗？

正面回答,当然没有这种可能.每个人的起点不一样,如果我的起始分数是0分,而其他人是5分,而6分就算成功.那我努力5分.他努力1分.乍一看,我的努力似乎并没什么卵用,这可能也是现在的父母都不想让自己的孩子输在起跑线上的原因吧.如果都是5分起跑,那你可能早就起飞了.然而现状是,你的起跑线要比你的比较对象低一截,甚至一大截.当然,你也可能会觉得,还好啊,就差一分,再加把劲也可以做到.可如果这最后一分,取决于运气,而你的运气又不是那么太好,又或者,在你的认知世界里,根本就没有6分的概念,而只有5分,你以为你已经到达了终点,可你却发现世界并不是你想象的样子,那你之前的努力,是否还有意义?

一切的努力,当然有意义.但是,我们需要切换参照系.在生活里我们总是习惯性的拿自己和他人进行比较.诚然,这没什么问题,我们不断成长的过程,实际上就是将更多的人不断比下去的过程.但是将他人比下去只是努力的结果,而不是努力的原因.我们不需要和他人进行比较,我们需要和自己进行比较.拥有5分的你,和只拥有1分的你,到底有什么区别.

举个例子,一个月后要期末考试,如果此时出现两个同样的我.假定一共有200道题目,努力的我,只能复习100题,不努力的我,完全不进行复习.但凡稍微有点逻辑能力的人都可以知道复习的我取得好成绩的概率要大于不复习的我.并且,这个结论是可以通过概率计算得出的.数学是一个很好的工具,可以指导我们的生活.然而生活有时候却喜欢开玩笑,一不小心,最后考试的题目只出了没有复习的100题.那似乎对于这次考试来看,你的努力似乎没什么改变什么现状.这又作何解释呢?

我们都知道,未来无法提前知晓,在这一个大前提下,如果我们什么都不做,当然也就什么都不会错.这也是绝大多数人的做法.可未来无法提前知晓,并不代表未来无法预测.我们依旧使用概率这个工具,来进行分析,如果只列公式,可能难以理解,那我们就假定有无穷多个平行空间,那200道题目,可以有多少个平行空间呢？

就是这么多个平行空间,怎么样,很庞大的一个数吧.在这么一个大的平行空间里,都是我没有复习过的题目的平行空间是多少呢?很明显,是1.也就是说,在这个计算题目中,我几乎可以肯定,复习的我有超过99.99%以上的概率要比不复习的我取的好成绩.那如果我们在一个月前就已经知道了这个结果,难道我们不就具备了对一个月后的考试结果的改变能力了么,也许会出现那几乎不可能出现的超低概率事件,可那又怎么样呢,生活并不是一次考试,我们无时无刻都在进行选择,当你想清楚这个思维的时候,实际上你就已经发生了变化,你已经从”不做就不错”的平行空间切换到了另一个对未来有那么一点点感知能力的平行空间.你再稍微多做一点,于是就跳向下一个.而此时,你的未来已经发生了变化.经常有朋友会跟我说,想的很明白,但是做不到,我的建议只有一个,想到的时候立马去做,哪怕只有五分钟,十分钟,半小时,只要有收获,实际上你的未来已经因此刻而发生变化.

当然每个人对未来的预测能力也是不一样的,这取决于一个人的各方面能力的总和.依旧举个例子,这时候就得和别人进行比较了.上学的时候老师都会划重点,但是每个人get到的信息是不一样的.学习好的同学总能很好的get到老师的要点,而学习不太理想但是努力的同学呢,可能就抓不到老师的点.就好比上面说的200道题目,可能学习好的同学会复习100个后考这些的概率会远大于你复习的那100个.又或者,你可能去复习200道题目以外的题目去了,最终你的结果看起来和没有复习是一样的.但是你可以因为这个例子就跟我说努力是自欺欺人吗?我觉得并不可以.而之所以会产生这种结果,是因为每个人学习能力,认知能力不同导致,而这些能力本身又是可以学习的.所以,还有什么好怕的呢,去追求自己想要的生活,不是什么自欺欺人,外界我们无法改变,但是我们可以在相同的外界环境下,改变自己,来取得我们想要的生活.

最后用一句非常俗套的话,努力,是为了成为更好的自己.实际上努力这个词,我觉得不是很贴切,我更喜欢成长这个词,我们不断的学习,实践,都是在完成一次次的成长.也可以说是一次次的跃迁,我们从一个本来不是那么太好的平行空间,跃迁到另一个稍微好一点的空间.实际上,在这一次次的选择中,你改变了你的未来.
