从脑科学的一个结论讲起

发表于 2017-02-26 | | 阅读次数

研究表明,我们的大脑终生都处于灵活状态.也可以说,大脑就像肌肉一样运作.因此直到老年,它都可以被深造.

通过这个研究,初步可以得出一个结论:聪明是可以学会的.这个结论对很多人来说冲击是很大的.无论你是谁,几乎都避免不了这个事实,总有那么一些人,能用你十分之一,甚至百分之一的时间,完成跟你一样的事情.于是很多人就觉得,自己不够聪明,也就找到了一个不断放弃自己的理由.我就是笨,做也做不好.然后心理暗示,恶性循环.友情提醒,不要小看暗示的力量.一切心理活动,都是生理活动.我没有否定天赋,我只是说,你可以变得更聪明.要用相对的角度,还记得吗.

之前有个朋友跟我说过这么一件事,他说高中毕业之后,大学假期无聊在家翻高中的数学书,发现很多以前学不明白的东西都很简单,自己还开开心心的做了两道题.他说的时候自顾自.可我却恍然大悟,实际上,高中毕业之后的这段时间,他的学习能力,是不断的再提升的.这也就是说,我们实际上就是在不断的变得聪明,这件事情从小就在发生,我们从小学,初中,高中,大学一直在学习,我们面对越来越多的困难,我们克服一个个的阻碍,我们依旧不和他人比较,我们使用相对的角度去观察,拥有小学知识水平的你,和拥有大学知识水平的你,哪个更聪明.如果到现在你还觉得聪明是天生就决定了的,那我换个词,哪个更智慧,或者说,哪个能力更强.毫无疑问,是拥有大学知识水平的你.那这也就说明了,我们本身是具备不断进化的能力的.只不过很多人,离开了学校,就没再继续做了.至于原因,可能是因为太苦了吧.毕业之后有多少人还会像在学校被老师看着的时候那样学习呢,下班回家葛优躺,玩玩游戏,洗洗睡觉不是挺好的吗?干嘛总是折磨自己,我不想变得更聪明了,或者说我想放慢我变聪明的速度,偷个懒,不行吗?

回答当然是可以,但是,我们来看一个表:

这是我今年看到的最有意思的一张表格,第一列是年数,第一行是年化收益,假定第一年,起始资金是1,那对应于不同的年化收益,达到资金为6的时间是不同的,10%的时候要19年,而如果是35%呢,只需要6年.哪怕只是多了5%的15%,也要比10%少了五年.而这中间的年化收益差距就你投资能力的差距,这也就可以类比聪明程度[智慧].你懂了吗?如果你能力强,你是可以缩短你达到财务目标的时间的,而这能力,前面说过了,是可以通过不断的学习,实践习得的.我觉得你可能还是没有看懂,我再把这个表格的内容翻译一次,

你是可以通过提升自己的聪明程度,让自己长寿的!

这个长寿,指的是相对值,可能你没有看懂,那我就继续解释一下,还是用上面那张表本身的含义吧.你如果想要达到资金6的位置,你通过提升自己的投资能力,可以缩短达到的年限不是吗?如果你初始值是10%(虽然99%的人并没有),你需要19年!可如果这19年中,你不断的进步,可能很快就跳到15%,甚至20%,你可能只用了10年.就达到了资金6的位置.相对于之前10%的那个你,你多了10年的时间,不仅如此,这10年时间里,你还拥有10%的那个你不曾拥有的财富.可以说,提升自己,不仅让你长寿,更让你拥有高质量的长寿.不仅如此,10%的人很多时候是理解不了20%的人做的决策的,因为超出了认知范围.不同的认知能力,带来的就是不同的世界观,价值观.最近看到一句话,”人和猪的差别远远没有人和人的差别大”.觉得挺有意思.你也可以观察一下生活,细细品味.

这个道理用在投资领域只是一个简单的例子,生活中方方面面很多地方都可以用到,如专业技能的培养,深入思考的能力,坚持锻炼身体等等.每个都不太那么容易,甚至说有时候会比较苦,可生活中,总会有那么一些人,他们似乎不断的在学习,不断的给自己找罪受,你看着他们难受,而他们呢,在享受呢.
